# Gorillax — pages légales (GitHub Pages)

Fichiers statiques pour App Store Connect (Privacy Policy + Terms of Service).

## Déploiement GitHub Pages

1. Créer un repo public `gorillax-legal` sur GitHub (compte `Arthur-destb38` ou autre).
2. Copier le contenu de ce dossier à la racine du repo.
3. **Settings → Pages → Build and deployment → Deploy from branch `main` / folder `/ (root)`**.
4. URLs publiques (exemple) :
   - `https://arthur-destb38.github.io/gorillax-legal/privacy.html`
   - `https://arthur-destb38.github.io/gorillax-legal/terms.html`
   - `https://arthur-destb38.github.io/gorillax-legal/privacy-en.html`
   - `https://arthur-destb38.github.io/gorillax-legal/terms-en.html`

## App Store Connect

Coller les URLs **privacy** et **terms** (FR ou EN selon la langue du listing ; Apple accepte souvent l'anglais pour le store international).
